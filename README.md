# 项目概述 python358

本项目为操作系统课程实验，采用 Python 语言实现了一个基于多级文件目录的文件管理系统。整个项目包含超过700行代码，配有详细注释和完整的报告文档。

# 技术栈

- Python 语言
- Python 标准库
- 第三方模块（文件操作、JSON处理）

# 功能模块

## 用户管理模块

- 用户注册
- 用户登录
- 用户注销

## 文件管理模块

### 文件模块

- 创建文件
- 打开文件
- 读取文件
- 写入文件
- 关闭文件
- 删除文件
- 重命名文件

### 文件目录模块

- 创建目录
- 切换目录
- 列出文件
- 解析路径

### 磁盘模块

- 保存磁盘数据

## 系统管理模块

- 显示帮助列表
- 退出系统
